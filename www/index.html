Investigation of dependencies between multiple data sources allows the discovery of regularities and interactions that are not seen in individual data sets. The importance of such methods is increasing with the availability and size of co-occurring data sets in computational biology, open data initiatives, and in other domains. Practical, open access implementations of general-purpose algorithms will help to realize the full potential of these information sources.

Your feedback and contributions are welcome. See the project page at R-Forge, or contact project admin.

Software & Documentation

The current version provides functionality for regularized dependency detection [2-3] and dependency-based dimensionality reduction (drCCA; [4]). In particular, the package provides an implementation of a general latent variable framework, which includes as special cases the probabilistic versions of PCA, factor analysis, and CCA, and their regularized variants.

The applicability of the models has been demonstrated in previous case studies [2-4]. See also our project page at R-Forge.

Background

Techniques for the discovery and analysis of statistical dependencies are implemented in R, an open source environment for statistical computing.

The implementations are based on well-established models such as probabilistic canonical correlation analysis and its variants [1-4]. Probabilistic framework deals rigorously with the uncertainties associated with small sample sizes, and allows incorporation of prior information in the analysis.

The modules will later be integrated into a unified set of tools for dependency modeling.

Installation

Each module has to be installed independently. For installation instructions and description of the modules, see the package vignette. Note that this is a development version.
